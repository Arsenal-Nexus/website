Overview

The Arsenal Nexus Mobile Base Station provides a reliable field hub for command, control, and data relay. Designed for rapid deployment, it aggregates ground links, backhauls over 4G/5G or SATCOM*, and coordinates fleets with built-in redundancy and power management. (*optional modules)

Key Features

  • Multi-radio support (ISM/LTE/5G; modular)
  • Automatic link failover & dual SIM
  • Rugged IP-rated enclosure
  • Integrated battery with hot-swap capability
  • Web dashboard for fleet status & logs
  • Edge compute slot for on-site processing
