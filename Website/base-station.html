Overview

The Arsenal Nexus Mobile Base Station provides a ruggedized, transportable communications hub that supports 5G, LTE, and satellite links. Designed for rapid deployment, it enables reliable command & control, telemetry, and data backhaul in remote or contested environments.

Key Features

  • Multi‑band cellular modem (4G/5G) + dual‑band satellite uplink
  • Integrated high‑gain directional antenna
  • Hot‑swap battery modules for up to 8 hours of operation
  • Built‑in VPN & firewall for secure connectivity
  • Rugged MIL‑STD‑810 chassis with IP67 sealing
